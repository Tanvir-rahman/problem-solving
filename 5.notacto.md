

**Input:**
```
2
.X.
.XX
X.X
...
.XX
...
```

**Output:**
```
Game #1: Bob
Game #2: Alice
```

**1st Turn Explanation:**
```
.X.
.XX
X.X
```
- In this game, there is a line for three X's in the second column.
- Alice (who made the last move) allowed this line of three X's, so she loses.
- Therefore, Bob wins the game.

**2nd Turn Explanation:**
```
...
.XX
...
```
- It's Bob's turn to play, and he can place an X in the empty cell in the second row to create a line of three X's.
- Since Bob is creating the line of X's, Alice wins this game.
